## Installation

```bash
$ npm install

# doppler install
$ brew install gnupg
$ brew install dopplerhq/cli/doppler

# doppler login
$ doppler login

# doppler setup
$ doppler setup
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
